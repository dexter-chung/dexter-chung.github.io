# Agile Project Management

The second thing that I learned from this course is Agile Project Management in which is the process of having a flexable approach to projects and being to adapt to changes when needed. When planning for a big project sometimes not ever aspect of that plan will come to fruition for example having to face time constraints. In those times, it is important to be able to prioritize what is actaully needed and what is not to be able to make the finished product. In Agile Project Management a project can be broken down into smaller simpler tasks which can help in organization.
